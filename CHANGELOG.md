# Change Log

## [1.0.0] - 2025-01-18

### Added
- Initial release of OSU Beatmap Syntax Highlighter
- Complete syntax highlighting for all OSU beatmap sections
- Support for timing points, hit objects, and storyboard commands
- Code folding and auto-closing pairs
- Comment support with `//` syntax
- File type recognition for `.osu` files

### Features
- Highlights section headers like [General], [Metadata], [TimingPoints]
- Color-codes key-value pairs with type-aware highlighting
- Recognizes numeric values, booleans, and file paths
- Supports complex hit object syntax including sliders and spinners
- Storyboard command highlighting with parameter recognition
- Works with all VSCode themes through semantic highlighting
